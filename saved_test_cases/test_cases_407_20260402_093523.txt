--- Batch 1: POSITIVE/HAPPY PATH SCENARIOS AND BASIC FUNCTIONALITY (Targeting TC-001 to TC-020) ---
ID: TC-001
Title: Valid Email Submission for Password Reset
Description: Verify that a valid email address submitted for password reset triggers a password reset email to be sent.
Preconditions: User is registered with a valid email address.
Steps:
1. User clicks the "Forgot Password" button on the login page.
2. User submits a valid registered email address in the email field.
3. System displays a generic success message.
Expected Result: A password reset email with a secure link is sent to the user's registered email address.
Priority: High
Test Type: positive/happy path scenarios and basic functionality

ID: TC-002
Title: Successful Password Reset with Valid Token
