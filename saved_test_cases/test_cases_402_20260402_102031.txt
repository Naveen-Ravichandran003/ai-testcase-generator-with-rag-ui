--- Batch 1: POSITIVE/HAPPY PATH SCENARIOS AND BASIC FUNCTIONALITY (Targeting TC-001 to TC-020) ---
ID: TC-001
Title: Successful Password Reset Request
Description: Verifies that a user can successfully request a password reset by submitting a registered email address.
Preconditions: User is registered with a valid email address.
Steps:
1. User clicks the "Forgot Password" button on the login page.
2. User submits a registered email address in the input field.
3. User clicks the "Submit" button.
Expected Result: The system displays a generic success message.
Priority: High
Test Type: positive/happy path scenarios and basic functionality
